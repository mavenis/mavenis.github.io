WEB для школьников

Этот курс предназначен для учащихся 7-10 классов, интересующихся web программированием и версткой сайтов, не знакомых с языком гипертекстовой разметки HTML и каскадными таблицами стилей CSS.

Цель курса - научиться создавать простые статичные сайты.

Курс дает возможность овладеть начальными навыками в изучении HTML и CSS.

По окончанию курса учащиеся овладеют HTML и CSS на начальном уровне.
